# grafos-av1
Cenário: Rede social (Direcionado e ponderado)

• Entrada: Lista de adjacência
• Primeira linha: n m tipo
– n: número de vértices (0..n-1)
– m: número de arestas/arcos
– tipo: D (direcionado) ou U (não-direcionado)
• Próximas m linhas: u v (opcional: w, se ponderado)

Aplicação de BFS e DFS com representação visual

Métodos complementares:
• Hubs (Top 3 mais conectados)
• Amigo a 2 passos (Lista de amigos do seu amigo)
